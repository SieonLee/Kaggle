# Student Performance Dataset

## Overview
Analysis of student performance data using machine learning techniques.

## Dataset
- **Source**: [Kaggle](https://www.kaggle.com)
- **File**: `Student_Performance_Dataset.ipynb`

## Requirements
```
pandas
numpy
matplotlib
seaborn
scikit-learn
```

## Usage
1. Clone the repository
2. Install dependencies
3. Open `Student_Performance_Dataset.ipynb` in Jupyter Notebook
